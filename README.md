# 🐦 BASED-FLAPPY

### Play & Earn mini app

A fully decentralized Flappy Bird game where players compete weekly for ETH prizes.

## ✨ Features
- 🔌 **Wallet Connect** - MetaMask, Coinbase Wallet, Trust Wallet (more integration is coming)
- 💰 **Entry Fee** - Only 0.00005 ETH
- 🏆 **Weekly Contest** - New contest every Monday
- 📊 **Global Leaderboard** - Real-time scores with Supabase
- 🎮 **Classic Gameplay** - Tap/Click to fly, avoid pipes

## 🎁 Prize Distribution
- 🥇 **1st Place** - 50% of total pool
- 🥈 **2nd Place** - 40% of total pool
- ⚡ **Platform Fee** - 10%

## 🚀 How to Play
1. Connect your Base wallet
2. Pay 0.00005 ETH entry fee
3. Play Flappy Bird
4. Submit your high score
5. Win ETH rewards weekly!

## 🛠️ Tech Stack
- HTML5 Canvas (Game)
- Supabase (Backend & Database)
- Ethers.js (Web3 Integration)
- Base Chain (Blockchain)
